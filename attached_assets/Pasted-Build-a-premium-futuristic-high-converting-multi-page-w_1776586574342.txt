Build a premium, futuristic, high-converting multi-page website for a digital agency called "Zentrix Solutions".

IMPORTANT:
This is a real business website. It must look polished, premium, trustworthy, and modern — not like a beginner template. The design should feel like a high-end digital agency / SaaS company.

CRITICAL LANGUAGE REQUIREMENT:
The website must support 2 languages:
- Dutch (default main language)
- English

There must be a visible language toggle in the navigation bar.
Default language on first load must be Dutch.

VERY IMPORTANT:
Every single text element on the website must be translatable with the toggle, including:
- navigation links
- hero headlines
- subheadlines
- buttons
- section titles
- paragraphs
- service card titles
- service card descriptions
- review section text
- CTA text
- contact form labels
- placeholders
- footer text
- small labels
- all page content
- all repeated/global elements

Do NOT leave any text hardcoded in only one language.
Use a clean translation object or dictionary in JavaScript so the entire site can switch between Dutch and English consistently.
The language toggle must update ALL visible text on ALL pages.
Use reusable keys for translations.
Store the selected language in localStorage so it stays active when navigating between pages.

BUSINESS POSITIONING:
Zentrix Solutions helps businesses grow with:
- Professional websites
- Custom applications
- Ongoing maintenance and optimization
- Transparent communication and reliable support

The site must clearly communicate:
- What Zentrix Solutions does
- Who Zentrix Solutions helps
- Why clients should trust the company
- That the company is transparent, modern, and results-driven

STYLE / VISUAL DIRECTION:
- Dark futuristic design
- Premium SaaS aesthetic
- Background colors: deep black / dark navy
- Accent colors: electric blue / soft blue glow
- White text with strong contrast
- Smooth hover effects
- Card-based layout
- Glassmorphism or subtle glow effects are welcome
- Clean spacing, elegant typography, minimal but powerful design
- Fully responsive on desktop, tablet, and mobile

TECH STACK:
- HTML, Tailwind CSS, and JavaScript
- Clean code structure
- Multiple pages
- Reusable navbar and footer
- Smooth scroll and polished transitions
- Use modern icons where relevant
- Everything must be functional and visually strong

CREATE THESE PAGES:
1. index.html
2. about.html
3. services.html
4. contact.html
5. styles.css (if needed)
6. script.js

If needed, also create a shared translations.js file or include a well-structured translation object inside script.js.

LANGUAGE TOGGLE REQUIREMENTS:
- Toggle visible in the top navigation bar
- Options: NL / EN
- Dutch must be selected by default
- On click, all text must switch immediately
- Selected language must persist across page navigation using localStorage
- All pages must load in the selected language automatically
- All content must use translation keys, not manually duplicated hardcoded sections
- Buttons, form labels, placeholders, footer items, cards, headings, and descriptions must all translate
