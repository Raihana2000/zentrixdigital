Gebruik Plan Mode eerst als die beschikbaar is. Doe geen brede SEO-herbouw. Fix alleen de sitemap.xml-problemen voor reldio.

Context:
- Website: https://www.reldio.eu
- Probleem: Google Search Console gaf bij sitemap.xml “Kan niet ophalen”.
- De sitemap bestaat en toont XML, maar moet volledig strak, consistent en Google-proof worden gemaakt.
- Bestaande routes, styling, taalfunctionaliteit, header/footer, componentstructuur en design tokens mogen niet worden gewijzigd.
- Geen package manager wijzigen.
- Geen nieuwe dependency toevoegen tenzij absoluut noodzakelijk.

Doel:
Maak /sitemap.xml volledig geldige, consistente en door Google ophaalbare XML.

Taken:

1. Zoek waar sitemap.xml wordt opgeslagen of gegenereerd.
   Controleer onder andere:
   - public/sitemap.xml
   - client/public/sitemap.xml
   - server routes
   - Vite/static asset config
   - eventuele custom sitemap generator
   - robots.txt verwijzing naar sitemap

2. Controleer dat /sitemap.xml live rechtstreeks pure XML oplevert:
   - geen HTML
   - geen React app shell
   - geen divs
   - geen scripts
   - geen browser/layout wrapper
   - geen extra nodes buiten <urlset>

3. Zorg dat de sitemap begint met exact:
   <?xml version="1.0" encoding="UTF-8"?>

4. Zorg dat de root exact een geldige sitemap namespace gebruikt:
   <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">

5. Controleer dat elke URL-entry alleen geldige sitemap-tags gebruikt:
   - <url>
   - <loc>
   - <lastmod> optioneel, maar bij voorkeur overal aanwezig

6. Maak alle URL’s consistent.
   Kies deze stijl:
   - homepage: https://www.reldio.eu/
   - alle andere routes: zonder trailing slash
   Voorbeeld:
   Goed: https://www.reldio.eu/websites
   Niet: https://www.reldio.eu/websites/

7. Voeg ontbrekende <lastmod> toe waar die ontbreekt.
   Gebruik voor bestaande oudere pagina’s 2026-04-26.
   Gebruik voor nieuwe/recente pagina’s 2026-05-02 of de meest logische bestaande datum uit de sitemap.
   Belangrijk: alle lastmod-waarden moeten geldig ISO-datumformaat hebben: YYYY-MM-DD.

10. Controleer robots.txt.
    Zorg dat robots.txt bereikbaar is op:
    /robots.txt

    En dat daarin minimaal staat:
    User-agent: *
    Allow: /

    Sitemap: https://www.reldio.eu/sitemap.xml

11. Controleer indien mogelijk de response headers van /sitemap.xml.
    De content-type moet application/xml, text/xml of een vergelijkbare XML-content-type zijn.
    Als de server/config dit verkeerd als text/html serveert, fix dit minimaal en veilig.

12. Voeg een simpele validatie toe zonder nieuwe dependency, bijvoorbeeld een klein script of command dat controleert:
    - sitemap bevat <?xml
    - sitemap bevat <urlset
    - sitemap bevat geen <html
    - sitemap bevat geen <div
    - sitemap bevat geen <script
    - alle <loc> bevatten https://www.reldio.eu
    - XML is syntactisch logisch opgebouwd

13. Schrijf een auditbestand:
    audit/sitemap-fix-summary.md

    Daarin moet staan:
    - waar sitemap.xml staat of hoe hij gegenereerd wordt
    - wat aangepast is
    - hoeveel URL’s erin staan
    - welke trailing slash strategie is gekozen
    - of robots.txt naar de sitemap verwijst
    - hoe is gecontroleerd dat er geen HTML/div/script in zit
    - welke build/verificatie is uitgevoerd
    - wat ik daarna handmatig in Google Search Console moet doen

14. Run bestaande verificatie:
    - gebruik bestaande package manager
    - run lint als script bestaat
    - run typecheck als script bestaat
    - run build als script bestaat
    - geen onnodige herhaalde runs

15. Commit de wijziging met:
    Fix Google Search Console sitemap XML

Acceptatiecriteria:
- /sitemap.xml is pure geldige XML.
- Geen <div>, <html>, <script> of React/app-shell in sitemap.
- Alle URL’s zijn absolute https://www.reldio.eu URL’s.
- URL-stijl is consistent: homepage met slash, andere routes zonder slash.
- Elke URL heeft een geldige lastmod.
- robots.txt verwijst naar https://www.reldio.eu/sitemap.xml.
- Build slaagt.
- audit/sitemap-fix-summary.md bestaat.
